Project for learning MQTT, NodeRed, influxDB and Grafana using Raspberry Pi 4 and ESP32.

Raspberry Pi 4:
- MQTT-broker
- NodeRed
  - IBM-cloud
  - Dashboard
    - Temperature
    - Humidity
    - LED ON/OFF
    - Slider for servomotor
    - Camerafeed from Raspi4 camera    
  - MQTT-publisher
  - MQTT-subscriber
- influxDB
- Grafana(?)

ESP32:
- DHT11-sensor
- Servomotor
- LED
- MQTT-publisher
- MQTT-subscriber
